📋 Klaviyo setup checklist for implementation
  • Smart sending: ON (Klaviyo default quiet-hours / recent-send suppression)
  • Initial delay: 45 minutes before Email 1 (so customers have time to settle after checkout — cancel filter not relied upon here)
  • No profile filters or exclusions set — flow fires on every Placed Order event. Standard Klaviyo suppressions (unsubscribed, bounce, spam) apply globally.
  • Entry conditional split: Evaluate branches in priority order — first match wins. Use the Items property from the Placed Order event (array of product name strings).
  • E1 founder image: Upload Elena's portrait to Klaviyo image library before launch
  • E1 Track Your Order CTA: Link to Shopify order status URL (Klaviyo has a dedicated merge tag for this)
  • E1 Generic dynamic product block: Requires Klaviyo Shopify catalog feed to be synced. Matches by product name from event.Items.
  • E2 Generic subject + body: Use {{ event.Items|first|default:"first Lohnr piece" }} with fallback
  • E4 free shipping code: Create FREESHIP-FIRST in Shopify — no minimum spend, static code, 30-day rolling expiry
  • E4 new arrivals block: Connect Shopify "New Arrivals" collection feed to Klaviyo, use a dynamic product block
  • E4 Facebook Group URL: [RO team to provide current Lohnr FB Group URL]
  • E5 plain-text rendering: Use Klaviyo's plain-text template — no hero, no logo, no styled footer. Standard unsubscribe block only.
  • E5 sender reply-to: Set to gabby@lohnr.com (or equivalent) so replies route to CX
